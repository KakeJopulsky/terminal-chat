# terminal-chat

*Lightweight command-line chat client built for my friends*

> ![screencap](https://i.imgur.com/4P4E2WT.png)

 The server is deployed to heroku, and can be *slow to start!*

## To run:

```
  npm install
  npm run client
```

Open as many client windows as you wish.

## Built with :

- Node / Express
- Socket.io
